# ココナラ出品支援

## 概要

ココナラに出品する商品の準備を支援します：
1. サムネイル画像の生成（Gemini 3 Pro Image使用）
2. 商品タイトルの作成
3. 商品説明文の作成
4. 出品用チェックリスト

## ワークフロー

### ステップ1: 商品情報のヒアリング

ユーザーに以下を確認：
- **サービス内容**: 何を提供するか
- **ターゲット**: 誰向けか
- **価格帯**: 想定価格
- **納期**: 作業日数
- **強み**: 他との差別化ポイント

### ステップ2: サムネイル画像の生成

`gemini-image` MCPツールの `generate_image` を使用してサムネイルを生成。

**推奨設定:**
- アスペクト比: `1:1`（正方形）
- 画像サイズ: `2K`（高解像度）
- 出力先: ユーザー指定のパス

**プロンプト作成のポイント:**
- サービス内容を視覚的に表現
- プロフェッショナルな印象
- 目を引くデザイン
- テキストは最小限（読みやすさ重視）

```
例: "Professional service banner for [サービス内容], modern minimalist design,
clean white background, subtle blue accents, professional and trustworthy atmosphere"
```

### ステップ3: 商品タイトルの作成

**タイトルのルール:**
- 最大40文字
- サービス内容が一目でわかる
- ターゲットを明確に
- 数字や具体性を入れる

**テンプレート:**
```
【{カテゴリ}】{サービス内容}を{納品物}でお届けします
{ターゲット}向け！{サービス内容}を{特徴}で提供
```

### ステップ4: 商品説明文の作成

**構成:**
```markdown
## このサービスについて
[サービスの概要を2-3行で]

## こんな方におすすめ
- [ターゲット1]
- [ターゲット2]
- [ターゲット3]

## サービス内容
[詳細な内容説明]

## 料金に含まれるもの
- [納品物1]
- [納品物2]

## 納期
基本: {X}日以内

## 購入にあたってのお願い
- [注意事項1]
- [注意事項2]

## 自己紹介
[簡単な自己紹介と実績]
```

### ステップ5: 出品前チェックリスト

- [ ] サムネイル画像は魅力的か
- [ ] タイトルは40文字以内か
- [ ] 説明文にサービス内容が明確に書かれているか
- [ ] 価格は適切か
- [ ] 納期は現実的か
- [ ] 注意事項・禁止事項は明記されているか

## 画像生成の使い方

MCPツール `generate_image` を使用：

```
ツール: generate_image
パラメータ:
- prompt: 画像の説明（英語推奨）
- output_path: 保存先パス
- aspect_ratio: "1:1"（ココナラ推奨）
- image_size: "2K"
```

## 出力例

最終的に以下を出力：

1. **サムネイル画像**: 指定パスに保存
2. **商品タイトル**: コピペ用テキスト
3. **商品説明文**: コピペ用テキスト
4. **カテゴリ提案**: ココナラのカテゴリ名

## 注意事項

- 画像生成にはGemini APIキーが必要
- 生成された画像は必ず確認してから使用
- 著作権・商標に注意
- ココナラの利用規約を遵守
